🔐 Admin Panel Setup Guide

Step 1: Run Migrations

Execute the following command to create the admin table and other profile-related tables:

php artisan migrate

Step 2: Seed Admin Accounts (Optional)

To create default admin accounts, run:

php artisan db:seed --class=AdminSeeder
📋 Default Admin Credentials:
  • Email: admin@astrology.com
  • Password: admin123
Or Moderator:

Step 3: Access Admin Panel

  • Open your browser and navigate to: http://localhost/astrology/admin/login
  • Enter the credentials above
  • Click "Login to Dashboard"

Step 4: Dashboard Features

  • ✓ View total user count
  • ✓ View total astrologer count
  • ✓ View pending astrologer approvals
  • ✓ View approved astrologers
  • ✓ View recent users (last 10)
  • ✓ View recent astrologers (last 10)
💡 Important Notes:
  • Change default admin passwords immediately after first login
  • Admin panel is protected by authentication middleware
  • All admin activities should be logged for security
  • The "Remember me" option persists login across sessions

API Endpoints Created

  • GET /admin/login - Admin login page
  • POST /admin/login - Process login
  • GET /admin/dashboard - Admin dashboard
  • POST /admin/logout - Logout admin

User Profile Update API (For Regular Users)

  • PUT /api/v1/user/profile/{userId} - Update user profile
  • Required fields: name, gender, date_of_birth, time_of_birth, place_of_birth, languages
  • User must verify OTP before updating profile
